Site Anime Dark — Shoto Todoroki

Um site inspirado no personagem Shoto Todoroki de My Hero Academia.

Sobre o projeto

Este projeto foi criado utilizando HTML e CSS com estilo dark/anime minimalista.

O objetivo do site é apresentar:

- imagens do personagem,
- visual moderno,
- páginas estilizadas,
- experiência adaptada para celular.

Tecnologias utilizadas

- HTML5
- CSS3
- GitHub Pages

Acesso ao site

O site está hospedado pelo GitHub Pages.
